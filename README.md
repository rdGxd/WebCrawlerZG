# Projeto: Web Crawler

## Descrição:
Este projeto tem como objetivo realizar a extração de informações de um site, utilizando a técnica de Web Crawler.

## Tecnologias:
- Groovy
- Gradle
- Jsoup
- HttpBuilder

## Execução:
Para executar o projeto, basta executar o comando abaixo:

1. Clone o repositório
2. Navegue a pasta `src/main/groovy/org/example`
3. Execute o ANSCrawler
